# ❤️ Heart Check (Streamlit App)

## 📌 About Project

Heart Check is a Streamlit web application that predicts the risk of heart disease based on user input. It uses a machine learning model to provide quick and simple health insights.

---

## 🛠️ Technologies Used

* Python
* Streamlit
* Pandas, NumPy
* Scikit-learn

---

## ⚙️ How to Run

1. Open the project folder

2. Install required libraries:
   pip install -r requirements.txt

3. Run the app:
   streamlit run app.py

---

## 📊 Working

* User enters health details
* Data is processed
* ML model predicts result
* Output is displayed on screen

---

## 📁 Files

* `app.py` – main Streamlit app
* `model.pkl` – trained ML model
* `requirements.txt` – dependencies
